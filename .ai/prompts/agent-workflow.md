# Agent Operating Procedure

Follow this procedure at the start of every session.

---

## Step 1 — Orient

1. Read `README.md` — project purpose and file structure
2. Read `.ai/README.md` — this folder

---

## Step 2 — Understand Code

| I need to… | Read… |
|---|---|
| Edit content or structure | `index.html` |
| Edit styling | `style.css` |
| Edit design tokens/reset | `base.css` |

---

## Step 3 — Work

Make targeted changes. Do not over-engineer — this is a static site.

**Common tasks:**
- Edit marketing content → `index.html`
- Add/change styles → `style.css`
- Adjust colors/spacing → `base.css` (tokens)
- Change themes → Look for theme classes in `style.css`

---

## Step 4 — Update Memory

Before finishing, if significant changes were made:
- Update `.ai/README.md` if file structure changed
